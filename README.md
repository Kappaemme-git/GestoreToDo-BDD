# Gestore ToDo - Database (PostgreSQL)

Database per il progetto “Gestore ToDo” (stile Trello) per il corso di Basi di Dati.

## Contenuto
- `sql/schema.sql`: schema completo (ENUM, tabelle, vincoli, trigger, funzioni e procedure)
- `sql/populate.sql`: dati di esempio per test
- `sql/dump_full.sql` (opzionale): dump completo generato da pgAdmin
